# homebrew-tap

Homebrew tap for `md2wechat`.

## Install

```bash
brew install geekjourneyx/tap/md2wechat
```

## Upgrade

```bash
brew upgrade geekjourneyx/tap/md2wechat
```

## Notes

- The current bootstrap formula points at the published `v2.0.2` release binaries so the tap is usable immediately.
- Starting with the next `md2wechat-skill` tag release, the source repository will auto-update `Formula/md2wechat.rb` to the tarball-based formula generated from the release bundle.
